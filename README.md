This repository contains implementations of several graph/tree search algorithms in C, including:
1. BMS (Exhaustic Search )
2. DFS
3. BFS
4. DFS+BFS
5. DFS + History
6. BFS + History
7. Hill climbing
8. Beam Search
9. HC + History
10. BS + History
11. Oracle
12. Branch & Bound Algorithm
13. Branch & Bound + Estimated Heuristic
14. A*
